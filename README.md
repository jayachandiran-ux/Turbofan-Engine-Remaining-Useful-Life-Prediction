# Turbofan Engine Remaining Useful Life Prediction

## 📌 Project Overview

This project predicts the Remaining Useful Life (RUL) of turbofan engines using the NASA CMAPSS FD001 dataset. A Random Forest Regressor was trained on engine sensor data to estimate the number of operational cycles remaining before engine failure. This project demonstrates the use of Machine Learning for Predictive Maintenance in the aerospace industry.

---

## 🎯 Objectives

- Predict the Remaining Useful Life (RUL) of turbofan engines.
- Analyze engine sensor measurements.
- Build a Random Forest Regression model.
- Evaluate model performance using regression metrics.

---

## 📂 Dataset

**Dataset:** NASA CMAPSS Turbofan Engine Degradation Simulation Data (FD001)

The dataset contains:
- Training data
- Test data
- Actual Remaining Useful Life (RUL) values
